## 🆘 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| Screenshots not saving | Use Snipping Tool (search in Windows) or OneDrive Snip |
| Git command fails | Make sure: `cd workshop_booking` first |
| Can't push to GitHub | Run: `git remote -v` to verify URL is correct |
| npm install takes too long | First time only - ~1 minute is normal |
| Can't login to React app | Make sure Django is running: `python manage.py runserver` |
| React app won't start | Run: `npm install` first, then `npm run dev` |
