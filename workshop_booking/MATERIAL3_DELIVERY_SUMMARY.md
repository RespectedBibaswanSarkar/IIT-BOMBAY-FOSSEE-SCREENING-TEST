## 🚀 How to Use

### Option 1: Just Use It (No Changes Needed)
The upgraded design is automatically applied to all pages using the base template.

### Option 2: Customize Colors
Edit CSS variables in `material3.css`:
```css
:root {
  --md-sys-color-primary: #006a4e;  /* Change this */
  /* ... other colors ... */
}
```

### Option 3: Add Dark Mode
Add dark theme CSS:
```css
@media (prefers-color-scheme: dark) {
  :root {
    --md-sys-color-background: #121212;
    /* ... dark colors ... */
  }
}
```

### Option 4: Create New Pages
Use the Material 3 components and CSS classes:
```html
{% extends 'workshop_app/base.html' %}

{% block content %}
<div class="container my-5">
    <h1>My Page</h1>
    <div class="card shadow-2">
        <div class="card-header">
            <h2 class="card-title">Title</h2>
        </div>
        <div class="card-body">Content</div>
    </div>
</div>
{% endblock %}
```
